## 📌 Sobre

O objetivo deste projeto é desenvolver uma aplicação WEB que permita fazer a gestão de tarefas de uma equipa de trabalho e que permita fazer time tracking das tarefas, de forma simples e com uma UI intuitiva. 

**Problema que resolve:**
- A maioria das aplicações disponíveis com estas funcionalidades no mercado, são demasiado complexas e não permitem importar as tarefas através de Excel ou criar recurrências para essas tarefas.  

**Público-alvo:**
- Equipas de trabalho pequenas, que necessitem de uma ferramenta intuitiva, simples e funcional para Task management e Time Tracking.


---

## ✨ Funcionalidades

- ✅ Criar Equipas, Projetos, Task lists e tarefas;
- ✅ Acompanhar o estado de cada tarefa;
- ✅ Alertas por email das tarefas que estão prestes a expirar;
- ✅ Controlar o tempo despendido em cada tarefa por cada colaborador;
- ✅ Gerir as tarefas por prioridade;
- 🔄 Dashboards e relatórios de tempo;
- 🚀 View em formato de calendário;
- 🚀 Possível integração com Microsoft To Do, Teams, Planner.
